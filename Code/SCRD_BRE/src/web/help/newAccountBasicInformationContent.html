This is the first step for creating account. When you access the Create New Account page, you will see the following page. Choose the "Form Submitted" first. Then fill in the text fields. Fields with an asterisk next to them are mandatory. Click the "Next" button when you're done. The "Cancel" button exits with neither saving the data nor creating the account.

If you make an error, the input with errors will become red. They will have an error icon next to them. Click on th icon to see the error message. Make sure everything input has no errors and click "Next" and you will go to Step 2
