The Notifications tab will show you the list of notifications. It will give you information about the payments you need to approve, the pending payment assigned to you. The notifications will generally contains a link to the relevant page. For example you can have a notification as "You have 4 payments in Approval Page" or "You have 2 records in Work Queue Page". Each time with the link.

In general, the notifications will be information which will require some further actions from you.
