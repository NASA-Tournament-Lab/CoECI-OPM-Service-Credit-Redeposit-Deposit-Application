You can access it on the account details page in the "Billing Summary" tab.

Click on the "Update Interest" button and a new popup will open.

The interest is calculated and the new invoice appears. The technician must print out this invoice and mail it to the claimant. It will not be printed by the nightly batch process. If the printer jams or the paper is lost, you can reprint this invoice on the same day that you updated the interest. The "Update Interest" button is still enabled so you can click it again. Clicking on it again brings up a new prompt. When you click "OK", the same invoice appears again because the program did not do another calculation.
